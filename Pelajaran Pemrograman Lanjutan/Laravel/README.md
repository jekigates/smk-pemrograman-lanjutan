### Instalasi

Untuk Windows

```
> cd nama-folder
> composer install
> cp .env.example .env
> php artisan migrate:fresh --seed
> php artisan key:generate
```

Pesan:
Jangan lupa buat database kosong dengan nama seperti folder yang mau diambil.

### Menjalankan Projek

Untuk Windows

```
> php artisan serve
```
